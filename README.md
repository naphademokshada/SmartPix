# 📸 SmartPix – Image Search Using Text

SmartPix is a simple and efficient web-based application that allows users to search and view images by entering keywords or text queries. The project uses the Unsplash API to fetch high-quality, royalty-free images in real-time.

## 🚀 Features

* 🔍 Search images using text input
* 🖼️ Display high-quality images instantly
* ⚡ Fast and responsive user interface
* 🌐 Integration with Unsplash API
* 📱 User-friendly design

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* Unsplash API

## 📌 How It Works

1. User enters a keyword in the search box
2. Application sends request to Unsplash API
3. Relevant images are fetched and displayed
4. User can browse multiple results

## 💡 Use Cases

* Finding images for projects or presentations
* Inspiration for design and creativity
* Learning API integration

## 🔧 Setup Instructions

1. Clone the repository
2. Open the project folder
3. Add your Unsplash Access Key in the JavaScript file
4. Run the project in a browser

## 📷 Output

Displays a grid of images related to the searched keyword.
